# Evaluation Rubric

## Scoring Criteria

### Flow Extraction (30%)
- **Complete (10)**: All major processing steps identified
- **Accurate (10)**: Steps correctly sequenced and described
- **Detailed (10)**: Sufficient technical detail provided

### Validation (40%)
- **Comprehensive (15)**: All relevant issues identified
- **Accurate (15)**: Issues correctly categorized and prioritized
- **Justified (10)**: Recommendations supported by evidence

### Reporting (20%)
- **Clear (7)**: Well-structured and readable
- **Complete (7)**: All required sections present
- **Actionable (6)**: Recommendations practical to implement

### Overall Quality (10%)
- **Consistency (5)**: Consistent application of standards
- **Professionalism (5)**: Appropriate tone and formatting

## Grade Scale

- **A (90-100)**: Excellent work, minor issues only
- **B (80-89)**: Good work, some areas for improvement
- **C (70-79)**: Satisfactory, significant issues present
- **D (60-69)**: Poor quality, major problems
- **F (0-59)**: Unacceptable, fundamental flaws

## Penalties

- Missing critical steps: -20 points
- Incorrect methodology assessment: -15 points
- Poor documentation: -10 points
